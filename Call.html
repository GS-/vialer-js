    Call

    Common Call class for Click-to-dial and WebRTC calling.

    Constructor

    new Call()

    Methods

    _incoming()

    Generic UI and state-related logic for an outgoing call. Note: first set the number and displayName in the parent, before calling this super.

    _outgoing()

    Generic UI and state-related logic for an outgoing call.

    _start(opts)

    Handle logic when a call is started; both incoming and outgoing.

    Parameters:
    Name Type Description
    opts Object

    Options to pass to _start.

    Properties
    Name Type Attributes Description
    timeout Number

    Postpones resetting the call state.
    force Boolean

    Force showing a notification.
    message String <optional>

    Force a notification message.

    _stop(opts)

    Takes care of returning to a state before the call was created. Make sure you set the final state of a call before calling cleanup. The timeout is meant to postpone resetting the state, so the user has a hint of what happened in between.

    Parameters:
    Name Type Description
    opts Object

    Options to pass to _stop.

    Properties
    Name Type Attributes Description
    force Boolean

    Force showing a notification.
    message String <optional>

    Force a notification message.
    timeout Number

    Postpones resetting the call state.

    setState(state)

    Keep the state local to this class, unless the call's id is known. Then we can keep track of the call from Vue.

    Parameters:
    Name Type Description
    state Object

    The state to update.
