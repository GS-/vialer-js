    Dialer~DialerModule

    The Dialer module. It takes care of actually dialing a phonenumber and updating the status about a call.

    Constructor

    new DialerModule()

    Methods

    addContextMenuItem()

    Add a right-click contextmenu item to all browser tabs.

    async dial(bNumber, tabopt)

    Calling this method with a provided tab will display the callstatus on top of that tab in an iframe. The poller is stopped whenever the callstatus dialog is closed or the returned status is final. Without a provided tab(like when called from the colleagues widget) there won't be a visible popup, but a html5 notification that is triggered when the status changes.

    Parameters:
    Name Type Attributes Description
    bNumber Number

    The number the user wants to call.
    tab Tab <optional>

    The tab from which the call was initialized, or null to use notifcations.

    showCallstatus(bNumber, status)

    A tab triggers this function to show a status dialog.

    Parameters:
    Name Type Description
    bNumber String

    Pass it asap to the callstatus page.
    status String

    Pass the initial status to the callstatus page.

    switchObserver(tab) → {Boolean}

    Called when the tab observer is initialized, by calling dialer:observer.ready on the background. Determines whether the DOM observer and c2d icons should be switched on or off. The callback is done to the observer script.

    Parameters:
    Name Type Description
    tab Object

    The tab that is requesting observer status.
    Returns:
    Boolean -
    • Whether the observer should be listening.
